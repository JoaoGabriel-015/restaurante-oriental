# Restaurante Oriental 🍣

Projeto fictício para avaliação de CSS - Tech4me

---

## Descrição do Projeto

Este é um site de restaurante oriental criado para uma avaliação de CSS, com foco em estilização, responsividade e animações. O projeto contém três páginas principais:

- **Página inicial (index.html)**  
  Logomarca estilizada em texto (primeira letra colorida e sombra), e link centralizado para acesso ao cardápio.

- **Página cardápio (cardapio.html)**  
  Exibe uma grade de produtos em cartões, com imagem, título, descrição e botão que leva à página de detalhes. Inclui barra de navegação Bootstrap, campo de busca e botão para voltar à página inicial.

- **Página detalhes (detalhes.html)**  
  Exibe detalhes do produto selecionado, com imagem, título, descrição, botão para realizar pedido, e formulário para cadastro de promoções, e animação de piscar no texto do botão comprar.

---

## Tecnologias Utilizadas

- HTML5
- CSS3 (com ênfase em estilização e animações)
- Bootstrap 4 (para a navbar e responsividade)
- Google Fonts (fonte Poppins)
